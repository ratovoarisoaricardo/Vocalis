## 🇬🇧 English

### 📖 About the Project
A sophisticated web-based voice assistant orchestrating voice recognition, speech synthesis, and AI processing via Gemini/Ollama APIs.

### ✨ Key Features
- 🎙️ Real-time speech recognition
- 🧠 Intelligent responses via Gemini & Ollama
- 🔊 Natural text-to-speech synthesis
- 🌐 Web-based interface

### 💻 Getting Started
To get a local copy up and running, execute the following commands:
```bash
start.bat
```

## 🇫🇷 Français

### 📖 À propos du projet
Un assistant vocal web sophistiqué orchestrant la reconnaissance vocale, la synthèse vocale et le traitement IA via les API Gemini/Ollama.

### ✨ Fonctionnalités clés
- 🎙️ Reconnaissance vocale en temps réel
- 🧠 Réponses intelligentes via Gemini & Ollama
- 🔊 Synthèse vocale naturelle (TTS)
- 🌐 Interface 100% web

### 💻 Démarrage rapide
Pour installer et lancer ce projet localement, exécutez ces commandes :
```bash
start.bat
```
